# Editor Configs
This repo contains configuration files for multiple text editors on multiple OS'es.

# Folder structure
The folders are divided per text editor, underneath the editor folder are subfolders per OS.<br>

example
``` text
├── doom_emacs
│   ├── macos
│   │   └── .doom.d
│   │       ├── config.el
│   │       ├── custom.el
│   │       ├── init.el
│   │       └── packages.el
│   └── windows
│       └── .doom.d
│           ├── config.el
│           ├── custom.el
│           ├── init.el
│           └── packages.el
└── vscode
    └── macos
        ├── keybindings.json
        └── settings.json
```
